# OpenRouter

> 💡 플러그인 버전 1.16.3 부터 지원

## 간략 설명

1.  공식 웹사이트: [OpenRouter](https://openrouter.ai/)
2.  공식 요금 정보: 무료 모델 제공, 자체 충전도 가능

## 사용 방법

1.  [OpenRouter](https://openrouter.ai/)를 열고 계정에 로그인합니다
2.  오른쪽 상단 아바타를 클릭하고 `Keys`를 선택한 다음 [페이지](https://openrouter.ai/settings/keys)로 이동하여 `Create API Key`를 클릭하여 API 키를 생성합니다
3.  API 키를 플러그인의 `API Key` 필드에 입력합니다
4.  완료 🎉, 질문이 있으시면 [여기](https://github.com/immersive-translate/immersive-translate/issues/137)에서 피드백을 제공해주세요.

## Come Usare

1.  Apri [OpenRouter](https://openrouter.ai/) ed effettua l'accesso al tuo account
2.  Fai clic sull'avatar nell'angolo in alto a destra, seleziona `Keys`, vai alla [pagina](https://openrouter.ai/settings/keys) e fai clic su `Create API Key` per creare una Chiave API
3.  Inserisci la Chiave API nel campo `API Key` del plugin
